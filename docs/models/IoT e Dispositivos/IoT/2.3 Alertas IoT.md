| #   | Método  | Rota                              | Descrição                                                     | Auth                                      | Fluxo                                        |
| --- | ------- | --------------------------------- | ------------------------------------------------------------- | ----------------------------------------- | -------------------------------------------- |
| A1  | `GET`   | `/iot/alertas`                    | Listar alertas com filtros `?estado=&tipo=&zona_id=&de=&ate=` | GESTOR, ADMIN | NestJS → PG réplica                          |
| A2  | `GET`   | `/iot/alertas/:id`                | Detalhe do alerta                                             | GESTOR, ADMIN                    | NestJS → PG réplica                          |
| A3  | `PATCH` | `/iot/alertas/:id/reconhecer`     | Operador reconhece o alerta                                   | GESTOR, ADMIN                    | NestJS → PG write reconhecido_por + em       |
| A4  | `PATCH` | `/iot/alertas/:id/resolver`       | Marcar como resolvido com nota                                | GESTOR, ADMIN                    | NestJS → PG write resolvido_em + notas       |
| A5  | `PATCH` | `/iot/alertas/:id/falso-positivo` | Marcar como falso positivo (calibração incorrecta)            | GESTOR, ADMIN                  | NestJS → PG write → feedback para calibração |
| A6  | `GET`   | `/iot/alertas/kpis`               | TMEF, MTTR, alertas por tipo/zona `?zona_id=&periodo=`        | GESTOR, ADMIN                  | FastAPI → Redis cache 15min ou PG agregação  |

**Corpo de A4 (resolver):**

```
{
  notas_resolucao: string (required),
  acao_tomada: "RECOLHA_EFECTUADA" | "SENSOR_REPARADO"
             | "CALIBRACAO_AJUSTADA" | "OUTRO" (optional)
}
```
