### Mapa de relacionamentos — domínio Cidadão/Utilizador

```
users
   │
   ├─── 1:1 ──→ cidadao_perfis           (WHERE role = 'CIDADAO')
   │               │
   │               ├─── N:1 ──→ zonas    (zona_residencia_id)
   │               │
   │               └─── índices: nif_hash, nome_completo (pg_trgm),
   │                             localizacao_ultima (GIST)
   │
   ├─── 1:1 ──→ gestor_perfis          (WHERE role IN
   │               │                        'GESTOR','ADMIN')
   │               │
   │               └─── GIN ──→ zonas    (zonas_responsabilidade UUID[])
   │
   ├─── 1:1 ──→ operador_perfis         (WHERE role = 'OPERADOR')
   │               │
   │               ├─── N:1 ──→ zonas    (zona_base_id)
   │               └─── N:M ──→ equipas_rota via operadores[] (terreno)
   │
   ├─── 1:N ──→ cidadao_consentimentos   (audit trail RGPD)
   │
   ├─── N:M ──→ ecopontos                via cidadao_ecopontos_favoritos
   │
   ├─── 1:1 ──→ cidadao_gamificacao      (estado gamificação)
   │               │
   │               ├─── 1:N ──→ quiz_sessoes
   │               └─── N:M ──→ badges   via cidadao_badges
   │
   ├─── 1:N ──→ reports                  (FK reports.cidadao_id → users.id)
   │
   ├─── 1:N ──→ pedidos_recolha          (FK pedidos_recolha.cidadao_id → users.id)
   │
   ├─── 1:N ──→ notificacoes             (FK notificacoes.cidadao_id → users.id)
   │
   ├─── 1:N ──→ partilhas_materiais      (FK partilhas_materiais.cidadao_id → users.id)
   │
   ├─── 1:N ──→ equipas_rota             (gestor_id; operadores[] → users.id)
   │               └─── 1:1 ──→ rotas_execucao (FK equipas_rota.rota_id)
   │
   └─── 1:N ──→ audit_log                (FK audit_log.actor_id → users.id)
```

**Nota:** Todas as FKs que antes apontavam para `cidadaos(id)` agora apontam para `users(id)`. Os nomes das colunas FK (como `cidadao_id`, `actor_id`) mantêm-se para preservar a semântica. As **rotas** já não referenciam um utilizador diretamente: a `rotas_execucao` liga-se à [[7.3 Schema PostgreSQL — equipas_rota|equipa_rota]], que agrega o(s) operador(es) e a carrinha. Ver [[07-Modelo-de-Dados]].
