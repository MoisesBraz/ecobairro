```
┌─────────────────────────────────────────────────────────────────────────────┐
│  tabela: rotas_execucao                                                     │
├──────────────────────────┬──────────────────────┬───────────────────────────┤
│ coluna                   │ tipo                 │ constraints / notas       │
├──────────────────────────┼──────────────────────┼───────────────────────────┤
│ id                       │ UUID                 │ PK                        │
│ zona_id                  │ UUID                 │ NOT NULL                  │
│                          │                      │ FK → zonas(id)            │
│ equipa_id                │ UUID                 │ NOT NULL                  │
│                          │                      │ FK → equipas_rota(id)     │
│                          │                      │ equipa que executa RF-29  │
│                          │                      │ (operador(es)+carrinha)   │
│ aceite_por               │ UUID                 │ NOT NULL                  │
│                          │                      │ FK → users(id)            │
│                          │                      │ operador que aceitou RF-30│
│ aceite_em                │ TIMESTAMPTZ          │ NOT NULL                  │
│                          │                      │ registo de quem aceitou   │
│                          │                      │ RF-05 obrigatório         │
│ ecopontos_planeados      │ UUID[]               │ NOT NULL                  │
│                          │                      │ ordem planeada da rota    │
│ ecopontos_visitados      │ UUID[]               │ NOT NULL, DEFAULT '{}'    │
│                          │                      │ preenchido durante exec.  │
│ geometria_rota           │ GEOGRAPHY            │ nullable                  │
│                          │                      │ (LINESTRING, 4326)        │
│                          │                      │ polilinha da rota         │
│ estado                   │ VARCHAR(20)          │ NOT NULL                  │
│                          │                      │ DEFAULT 'ACEITE'          │
│                          │                      │ CHECK IN (                │
│                          │                      │  'ACEITE',                │
│                          │                      │  'EM_CURSO',              │
│                          │                      │  'CONCLUIDA',             │
│                          │                      │  'CANCELADA')             │
│ iniciada_em              │ TIMESTAMPTZ          │ nullable                  │
│ concluida_em             │ TIMESTAMPTZ          │ nullable                  │
│ notas                    │ TEXT                 │ nullable                  │
│ criado_em                │ TIMESTAMPTZ          │ NOT NULL, DEFAULT now()   │
└──────────────────────────┴──────────────────────┴───────────────────────────┘

Índices:
  (equipa_id)                      BTREE — rota da equipa (1:1)
  (aceite_por, criado_em DESC)     BTREE — histórico do operador
  (zona_id, estado, criado_em)     BTREE — gestão por zona
  (estado) PARTIAL WHERE           BTREE — rotas activas
    estado IN ('ACEITE','EM_CURSO')
```

> **Retificação (RF-29/RF-30):** a rota deixou de referenciar `operador_id` diretamente. Passa a ligar-se a uma [[7.3 Schema PostgreSQL — equipas_rota|equipa de rota]] (`equipa_id`), que agrega o(s) operador(es) + a [[7.2 Schema PostgreSQL — carrinhas|carrinha]]. `aceite_por` regista o operador que aceitou a execução. O **Gestor** cria a equipa; o **Operador** executa (ver [[Execução de Rota (Operador)]]).

```
rotas_execucao (N:1) zonas
rotas_execucao (1:1) equipas_rota   FK: rotas_execucao.equipa_id
rotas_execucao (N:1) users          FK: rotas_execucao.aceite_por (operador)
rotas_execucao referencia (N) ecopontos via ecopontos_planeados[]
                                          e ecopontos_visitados[]
```
